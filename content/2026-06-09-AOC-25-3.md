## Part 1

Given a list of battery banks represented as strings of digits, identify the two digits within each bank that form the highest possible numerical value while maintaining their original relative order, then calculate the sum of these maximum joltages across all banks.

## Part 2

Given the same list of battery banks, identify the twelve digits within each bank that form the highest possible 12-digit numerical value while maintaining their original relative order, then calculate the total sum of these larger maximum joltages across all banks.

## Example input

```
987654321111111
811111111111119
234234234234278
818181911112111
```

## Solution

First, structs to store state and information in are defined. Each line is taken and turned into a slice of integers, `BatteryRow`, which will be rearranged. Row Simulation keeps the state of this rearrangement. Simulation keeps state for the whole problem.

```
type BatteryRow struct {
	readings []int
}

type RowSimulation struct {
	input              []int
	numbersLeft        int
	slots              []int
	initialSlotsLength int
}

type Simulation struct {
	sum         int
	simulations []RowSimulation
}

func NewRowSimulation(row BatteryRow, joltageLength int) RowSimulation {

	initialSlots := make([]int, joltageLength)

	return RowSimulation{
		input:              row.readings,
		numbersLeft:        len(row.readings),
		slots:              initialSlots,
		initialSlotsLength: joltageLength,
	}
}

func NewSimulation(rowSimulations []RowSimulation) Simulation {
	return Simulation{
		sum:         0,
		simulations: rowSimulations,
	}
}
```

Each line is parsed into `BatteryRow`.

Note that `int(r-'0')` is used as this converts runes into integers.

```
func ParseData(filename string) (instructions []BatteryRow, err error) {
	file, err := os.Open(filename)
	if err != nil {
		return nil, fmt.Errorf("Failed to open file: %v", err)
	}
	defer file.Close()

	scanner := bufio.NewScanner(file)

	for scanner.Scan() {
		line := scanner.Text()

		row := BatteryRow{
			readings: make([]int, 0, len(line)),
		}

		for _, r := range line {
			row.readings = append(row.readings, int(r-'0'))
		}

		instructions = append(instructions, row)
	}
	if err := scanner.Err(); err != nil {
		return nil, fmt.Errorf("Scanning error")
	}
	return instructions, nil
}
```

The following logic is then used to find the answer for each line:

Create a slice of zeros to store answers in. Take each number from the battery line one by one. Starting at index zero compare them with the numbers in the slice. If it is bigger, replace the current number with that number and then change every following number in the slice to zero.

A constraint is needed to ensure that the slice ends with no zero values remaining. The condition `attempt.NumbersRemaining >= attempt.SlotsLength-index-1` ensures that numbers are at the end of battery lines cannot replace numbers in the slice that are too far away. If they could replace these values, zeroes would be left in the final answer.

The total for each row is calculated and then added to `Simulation`'s overall state.

```
func (s *Simulation) Run() {
	for _, sim := range s.simulations {
		for _, digit := range sim.input {
			sim.numbersLeft--
			replaced := false

			for index, slot := range sim.slots {
				if replaced {
					sim.slots[index] = 0
				}

				if digit > slot && sim.numbersLeft >= sim.initialSlotsLength-index-1 && !replaced {
					sim.slots[index] = digit
					replaced = true
				}
			}
		}
		rowValue := 0
		for _, n := range sim.slots {
			rowValue = (rowValue * 10) + n
		}
		s.sum += rowValue
	}
}
```

The function `main` orchestrates the process. Loops through the parsed data to create structs for each row adding them to a slice which is then used to build the simulation.

```
func main() {

	joltageLengths := []int{2, 12}

	for _, length := range joltageLengths {

		joltageLength := length

		data, err := ParseData("input.txt")
		if err != nil {
			fmt.Println(err)
		}

		rowSimulations := make([]RowSimulation, 0, len(data))

		for _, d := range data {
			rowSim := NewRowSimulation(d, joltageLength)
			rowSimulations = append(rowSimulations, rowSim)
		}

		sim := NewSimulation(rowSimulations)

		sim.Run()

		fmt.Printf("The answer when the joltage output needs %d digits is %d\n", length, sim.sum)
	}
}
```
