This is my [_solution_](https://github.com/dnlaxe/advent-of-code/tree/main/2025/3) to [_AOC Day 4_](https://adventofcode.com/2025/day/4).

## Part 1

Given a grid of paper roll locations, scan for rolls that have fewer than four neighbors in the eight adjacent positions, and calculate the total count of these accessible rolls.

## Part 2

Given the same grid, simulate an iterative removal process where newly accessible rolls are removed until no more rolls meet the accessibility criteria, and calculate the total number of rolls removed throughout the entire process.

## Example input

```
..@@.@@@@.
@@@.@.@.@@
@@@@@.@.@@
@.@@@@..@.
@@.@@@@.@@
.@@@@@@@.@
.@.@.@.@@@
@.@@@.@@@@

```

## Solution

First, create structs for the grid of positions and each position to simplify what the interfaces need.

```
grid.go

type Position struct {
	Row int
	Col int
}

type Grid struct {
	Map map[Position]struct{}
}

func NewGrid() Grid {
	return Grid{
		Map: make(map[Position]struct{}),
	}
}

func (g Grid) IsOccupied(p Position) bool {
	_, exists := g.Map[p]
	return exists
}

func (g Grid) Remove(p Position) {
	delete(g.Map, p)
}

func (g Grid) CountAdjacent(p Position) int {
	count := 0
	for _, adj := range NewAdjacentPositions(p) {
		if g.IsOccupied((adj)) {
			count++
		}
	}
	return count
}

func NewAdjacentPositions(p Position) []Position {
	return []Position{
		{Row: p.Row - 1, Col: p.Col - 1}, {Row: p.Row - 1, Col: p.Col}, {Row: p.Row - 1, Col: p.Col + 1},
		{Row: p.Row, Col: p.Col - 1}, {Row: p.Row, Col: p.Col + 1},
		{Row: p.Row + 1, Col: p.Col - 1}, {Row: p.Row + 1, Col: p.Col}, {Row: p.Row + 1, Col: p.Col + 1},
	}
}
```

A seperate file `simulation.go` is created to handle the logic.

The logic:

Create a map in which each key is a position that contains a roll of paper. The roll of paper's existence is all that is needed, so the value can be `struct{}{}`. This is a useful way to fill the key's values as `struct{}{}` contains zero bytes.

This map is then iterated through. The surrounding positions' co-ordinates for each position are calculated (`NewAdjacentPositions`) and counted (`CountAdjacent`) if it exists in the map.In this example, the roll needs less than four non-empty spaces around it to be removed. If it can be removed, it's position is added to `toBeRemoved`.

After all positions have been iterated through, a record is recorded in the simulation's history and the positions collected in `toBeRemoved` are each removed from the map. While not explicitly stating so in the question, it is implied through the example that rolls shouldn't be removed during the round. A message is then printed with information about how many rolls were removed in that round.

This process is then repeated with the updated map as another round. This will continue as part of a while loop until there are no positions to be removed.

```
simulation.go

type Simulation struct {
	Grid         Grid
	History      map[int]int
	RemovedRolls int
	SpaceNeeded  int
}

func NewSimulation(grid Grid, spaceNeeded int) Simulation {
	return Simulation{
		Grid:         grid,
		History:      make(map[int]int),
		RemovedRolls: 0,
		SpaceNeeded:  spaceNeeded,
	}
}

func (s *Simulation) Run() {
	round := 1
	for {
		toBeRemoved := []Position{}

		for pos := range s.Grid.Map {

			if s.Grid.CountAdjacent(pos) < s.SpaceNeeded {
				toBeRemoved = append(toBeRemoved, pos)
			}
		}

		if len(toBeRemoved) == 0 {
			break
		}

		s.History[round] = len(toBeRemoved)

		for _, pos := range toBeRemoved {
			s.Grid.Remove(pos)
			s.RemovedRolls++
		}

		count := s.History[round]
		unit := "rolls"
		if count == 1 {
			unit = "roll"
		}
		fmt.Printf("%d %s rolls were removed in round %d\n", count, unit, round)
		round++
	}
}
```

The file `main.go` controls the program. It parses the data into a useable map and then passes the map into a simulation before printing the end result.

```
func main() {

	spaceNeeded := 4

	grid, err := parseDataIntoMap("input.txt")
	if err != nil {
		fmt.Println(err)
	}

	sim := NewSimulation(grid, spaceNeeded)

	sim.Run()

	fmt.Printf("%d rolls were removed in total\n", sim.RemovedRolls)
}

func parseDataIntoMap(filename string) (g Grid, err error) {

	file, err := os.Open(filename)
	if err != nil {
		return Grid{}, err
	}
	defer file.Close()

	g = NewGrid()

	scanner := bufio.NewScanner(file)

	row := 0

	for scanner.Scan() {
		line := scanner.Text()

		for idx, pos := range line {
			if pos == '@' {
				g.Map[Position{Row: row, Col: idx}] = struct{}{}
			}
		}
		row++
	}
	if err := scanner.Err(); err != nil {
		return Grid{}, fmt.Errorf("Scanning error")
	}
	return g, nil
}
```
