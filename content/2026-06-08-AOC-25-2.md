## Part 1

Given a list of comma-separated number ranges, scan for invalid IDs formed by a sequence of digits repeated exactly twice (such as 55 or 123123) with no leading zeroes, and calculate the total sum of these numbers.

## Part 2

Given a list of comma-separated number ranges, scan for invalid IDs formed by a sequence of digits repeated at least twice (such as 12341234 or 111) with no leading zeroes, and calculate the total sum of these numbers.

## Solution

First, structs to store state and information in are defined.

```
type Range struct {
	Start int
	End   int
}

type Simulation struct {
	Ranges                       []Range
	twoRepeatingPartsSum         int
	moreThanTwoRepeatingPartsSum int
	primeCache                   map[int][]int
}

func NewRange(start int, end int) Range {
	return Range{
		Start: start,
		End:   end,
	}
}

func NewSimulation(parsedRanges []Range) *Simulation {
	return &Simulation{
		Ranges:                       parsedRanges,
		twoRepeatingPartsSum:         0,
		moreThanTwoRepeatingPartsSum: 0,
		primeCache:                   make(map[int][]int),
	}
}
```

Next, a function to parse the data is created to turn data from text file into a slice of Range structs `[]Range{{Start: 11, End: 22},{Start: 95, End: 115},{Start: 998, End: 1012}}`

```
func ParseInstructions(filename string) (ranges []Range, err error) {

	file, err := os.ReadFile(filename)
	if err != nil {
		return nil, fmt.Errorf("Failed to open file: %v", err)
	}

	fileString := string(file)
	stringRanges := strings.Split(fileString, ",")

	for _, r := range stringRanges {
		pair := strings.Split(r, "-")

		start, _ := strconv.Atoi(pair[0])
		end, _ := strconv.Atoi(pair[1])

		newRange := NewRange(start, end)

		ranges = append(ranges, newRange)
	}

	return ranges, nil
}
```

The implementation logic is then defined calling on several helpers: `getPrimeFactors()`, `splitStringsIntoParts()`, and `allPartsAreEqual()`. Take one range from Ranges at a time and iterate through each value. The number of times a number can be sliced into equal parts is found by finding the prime factors of its length. The number is then sliced in a loop by these numbers and its slices are compared. If they are equal, the number is added to the total sum and the loop breaks.

```
func (s *Simulation) Run() {
	for _, r := range s.Ranges {

		for i := r.Start; i <= r.End; i++ {

			num := strconv.Itoa(i)
			numLength := len(num)

			factors, found := s.primeCache[numLength]
			if !found {
				factors = getPrimeFactors(numLength)
				s.primeCache[numLength] = factors
			}

			for _, factor := range factors {
				segments := splitStringIntoParts(num, factor)

				if areAllPartsEqual(segments) {
					s.moreThanTwoRepeatingPartsSum += i
					if factor == 2 {
						s.twoRepeatingPartsSum += i
					}
					break
				}
			}
		}
	}
}
```

As many numbers share the same length, a `primeCache` is used to remember calculated values. For a range of 1,000,000 numbers (all 6 digits long), `getPrimeFactors(6)` would need to be called 1,000,000 times. With this cache, that calculations will happen just once.

factors, found := s.primeCache[numLength]
if !found {
factors = getPrimeFactors(numLength)
s.primeCache[numLength] = factors
}

The `main` function parses the data then uses it to instantiate the simulation, before running it and printing the results.

```
func main() {
	parsedRanges, err := ParseInstructions("input.txt")
	if err != nil {
		log.Fatalf("Fatal: Failed to read input file: %v\n", err)
	}

	sim := NewSimulation(parsedRanges)

	sim.Run()

	fmt.Printf("Total of invalid IDs with two repeating parts: %d\n", sim.twoRepeatingPartsSum)
	fmt.Printf("Total of invalid IDs with two or more repeating parts: %d\n", sim.moreThanTwoRepeatingPartsSum)
	fmt.Printf("Prime Cache: %v\n", sim.primeCache)
}
```

A helper is used to find the prime factors. First the number is checked to see it is even, 2 is added to the returning slice, and it is then continued to be halved until `num%2 == 0` isn't true. This ensures 2 is only added to the returning slice once. A similar process then follows iterating through odd numbers to the point of exhaustion. If a remainder remains and it is greater than 1, it cannot have any smaller factors and therefore is a prime number.

```
func getPrimeFactors(num int) (primeFactors []int) {

	if num%2 == 0 {
		primeFactors = append(primeFactors, 2)
		for num%2 == 0 {
			num /= 2
		}
	}

	for i := 3; i*i <= num; i += 2 {
		if num%i == 0 {
			primeFactors = append(primeFactors, i)
			for num%i == 0 {
				num /= i
			}
		}
	}

	if num > 1 {
		primeFactors = append(primeFactors, num)
	}

	return primeFactors
}
```

Two more helpers are then used to split and look for matching equal parts.

```
func splitStringIntoParts(word string, parts int) (segments []string) {
	length := len(word)
	segmentLength := length / parts
	for i := range parts {
		start := i * segmentLength
		end := start + segmentLength
		segments = append(segments, word[start:end])
	}
	return segments
}

func areAllPartsEqual(segments []string) bool {
	reference := segments[0]

	for i := 1; i < len(segments); i++ {
		if segments[i] != reference {
			return false
		}
	}

	fmt.Println("Found: ", segments)
	return true
}
```
